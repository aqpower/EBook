# EasyBook-Web

## 食用方法

1. 安装依赖
```
pnpm install
```

2. 启动项目，命令参数配置可以参考`package.json`文件
```
pnpm run dev
```

## 使用的依赖
pinia持久化
```
pnpm i pinia-plugin-persistedstate
```
headlessUI库
```
pnpm install @headlessui/vue
```
daisyUI库
```
pnpm i -D daisyui@latest
```
tailwindcss的文本类
```
install -D @tailwindcss/typography
```
Icon图标库
```
pnpm install --save-dev @iconify/vue
```
支持Vue3的事件总线
```
pnpm i mitt
```
